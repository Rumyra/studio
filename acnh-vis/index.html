Animal Crossing Dataviz.

August 2022

I've played around with d3 a number of times, but never taken the time out to do any formal learning. With my new job and my new puppy, I've found I don't code so much and I'm also pretty housebound (for the time being), so this led me to start a d3 course.

I remember when I was in Australia speaking at JSConf Auz, I spent some time with Nadieh Brener, who, at the time, was working on 24 dataviz with Shirley Wu. It was a really inspiring project, the dataviz were amazing and the idea of doing something once a month has stuck with me.

Anyway, I wanted to start doing some dataviz projects which were personal and fun. I found an api for Animal Crossing data, like many others, my lockdown game.
